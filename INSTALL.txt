SetaMac — install

1. Unzip and move SetaMac.app to Applications.

2. Open it
   Right-click SetaMac.app → Open → Open.

   Says "damaged"?
   - Cancel
   - ⌘ Space → type Terminal → Return
   - Paste: xattr -cr /Applications/SetaMac.app
   - Return, then open SetaMac again

3. In SetaMac
   Start setup → add folders → Rescan library.

More help: github.com/manupastorr/seta-mac/blob/main/docs/DOWNLOAD.md
